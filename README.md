# QR-code-generator

#### Description:

This repository contains the source code for a QR code generation project. Built with HTML, CSS, and JavaScript, this project allows users to generate QR codes dynamically based on input data, making it a useful tool for encoding URLs, text, and other information into QR codes.

#### Features:

- Clean and modern design
- Responsive layout for mobile and desktop devices
- Real-time QR code generation
- Customizable QR code size and error correction levels
- Easy integration into existing projects
